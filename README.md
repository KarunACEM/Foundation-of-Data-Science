# Data Visualization Using Health Dataset

## Objective
To analyze and visualize health-related data using different data visualization techniques.

## Dataset
Health_heart_experimental.csv

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code
- Jupyter Notebook

## Visualizations Included
1. Line Chart – Average Heart Rate vs Age
2. Bar Plot – Average Weight by Health Indication
3. Pie Chart – Distribution of Health Indication
4. Box Plot – Diastolic Blood Pressure by Indication
5. Histogram – Height Distribution
6. Scatter Plot – Relationship Between Weight and Heart Rate

## Conclusion
Different visualization techniques were used to explore trends, distributions, and relationships among health variables. The analysis provides insights into cardiovascular and physical attributes of individuals.
